# 人工智能科创实践套装

这是一个专为人工智能科创实践套装开发的 Mind+ 图形化编程扩展库。通过本扩展，你可以轻松控制套装内各种传感器、电机和舵机。

![alt text](arduinoC/_images/featured.png)

## 功能模块

### 1. 输入模块 (Input)
- **数字输入 (Digital Input)**
  - 支持模块：按键 (Button)、人体红外传感器 (Pyroelectric Sensor) 等。
  - 功能：读取数字信号。对于按键模块，库内部会自动处理反逻辑（按下为高电平）。
- **模拟输入 (Analog Input)**
  - 支持模块：电位器 (Potentiometer)、土壤湿度传感器 (Soil Humidity) 等。
  - 功能：读取模拟信号数值。
- **超声波测距 (Ultrasonic)**
  - 功能：通过 I2C 接口读取前方障碍物的距离（单位：厘米）。

### 2. 输出模块 (Output)
- **数字输出 (Digital Output)**
  - 支持模块：LED 灯等。
  - 功能：控制引脚的高低电平输出。
- **LED 亮度控制 (LED Brightness)**
  - 功能：通过 PWM 控制 LED 的亮度（范围：0-100%）。

### 3. 电机控制 (Motor)
- **直流电机 (DC Motor)**
  - 端口：M1, M2, M3, M4 或全部 (All)。
  - 控制：设置速度 (0-255) 和方向 (正转 CW / 反转 CCW)。
- **电机停止**
  - 功能：停止指定端口的电机转动。

### 4. 舵机控制 (Servo)
- **270° 舵机**
  - 端口：S1 - S8。
  - 控制：设置舵机角度 (0-270°)。
- **360° 舵机**
  - 端口：S1 - S8。
  - 控制：设置舵机角度 (0-360°)。

## 注意事项
- 确保主控板K10已正确连接扩展板。
- 使用超声波传感器时，请确保接在正确的 I2C 接口上。
